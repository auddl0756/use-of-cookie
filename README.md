## 간단 정리
- 쿠키 = 작은 정보
- 쿠키 = (key,value)
- 쿠키는 서버로 재요청 시 자동으로 전송된다고 함..
- 쿠키 관리는 클라이언트측에서 함. <br/>
  서버에서 setMaxAge(0)을 하는 것은 클라이언트에 간접적으로 삭제를 요청하는 것이다.
- 크롬 개발자 도구의 Application 탭에서 쿠키들 볼 수 있음
